## 核心計算一：五行強度矩陣（Method C 基礎）

> 所有章節計算的根源，先建立命局五行強度分布。

### 步驟 1：基礎點數賦值

```
八字共 8 字，合計基礎 100 點，依以下比例分配：

月支：30 點（權重最重，占命局 30%）
  月支藏干分配（依本中餘比例 5:3:2，合計 30）：
    本氣：15 點
    中氣：9 點
    餘氣：6 點
  僅有本氣（如子→癸）：本氣 30 點

其他 7 字：各 10 點（合計 70 點，每字占 10%）
  天干（年干/月干/日干/時干）：直接歸屬該干五行 10 點
  地支（年支/日支/時支）：藏干按本中餘比例 5:3:2 分配（合計 10）：
    本氣：5 點
    中氣：3 點
    餘氣：2 點
  僅有本氣（如子→癸）：本氣 10 點

注：日干代表命主本身，其 10 點仍依五行歸屬計入（決定日主五行強度）
```

### 步驟 2：旺相休囚死乘數（依月支季節）

```
五行旺相休囚死對照表：
         木     火     土     金     水
春(寅卯辰) 旺×1.8 相×1.3 死×0.2 囚×0.5 休×1.0
夏(巳午未) 休×1.0 旺×1.8 相×1.3 死×0.2 囚×0.5
秋(申酉戌) 死×0.2 囚×0.5 休×1.0 旺×1.8 相×1.3
冬(亥子丑) 相×1.3 死×0.2 囚×0.5 休×1.0 旺×1.8
四季土月   囚×0.5 休×1.0 旺×1.8 相×1.3 死×0.2
（辰戌丑未）

注：月支藏干已由步驟1賦予 30 點基礎，此乘數直接套用於月支各藏干點數
```

### 步驟 3：地支互動調整

```
計算順序（由強至弱依序套用）：
1. 先判斷三會（最強）：命局四支中有三會 → 三會五行點數 ×2.5
2. 再判斷三合：命局四支中有三合 → 三合五行點數 ×2.0
3. 再判斷六合：六合對 → 本氣 ×1.10
4. 再判斷六沖：被沖 → 本氣 ×0.50
5. 再判斷三刑：被刑 → 本氣 ×0.60
6. 再判斷六害：被害 → 本氣 ×0.70
7. 最後六破：被破 → 本氣 ×0.75

注：同一地支若同時被多種互動影響，乘數疊加計算
    例：某支同時六沖且六害 → ×0.50 × 0.70 = ×0.35
```

### 步驟 4：計算五行比例

```
木總分 = Σ(所有屬木字符之調整後點數)
火總分 = Σ(所有屬火字符之調整後點數)
土總分 = Σ(所有屬土字符之調整後點數)
金總分 = Σ(所有屬金字符之調整後點數)
水總分 = Σ(所有屬水字符之調整後點數)

各五行% = 各五行總分 / (五種五行總分之和) × 100
（輸出：木X% 火X% 土X% 金X% 水X%，合計100%）
```

## 核心計算二：日主強弱百分制（Method A）

### 計算公式

```
依日干五行，將五行分為兩陣營：

比印陣（幫身）：
  日干同五行（比劫）點數 + 日干被生五行（印星）點數

洩克陣（耗身）：
  日干所生五行（食傷）點數
  + 日干所克五行（財星）點數
  + 克日干五行（官殺）點數

身強度 = 比印陣總分 / (比印陣 + 洩克陣) × 100
```

### 判斷閾值

```
≥ 70 → 身強（極強）
60-69 → 身強
45-59 → 中和
35-44 → 身弱
< 35  → 身弱（極弱）

極強（≥ 80 且無官殺制） → 評估從旺格
極弱（≤ 20 且無印比）   → 評估從強格
```

### 輸出格式

```
【日主強弱】日干X（五行），月令{得令/失令}（MOON_WEIGHT=1.5），
五行分布：木X% 火X% 土X% 金X% 水X%
比印陣：X% ｜ 洩克陣：X%
地支互動：{三合/三會/六沖/三刑/六害/六破 影響說明}
結論：日主{身強/中和/身弱}（強弱度：X%）
```

## 第二章：命局體性（寒暖濕燥）

> 完全依月支判定，其他干支不影響體性結論。

### 輸入
- 月支(MB)

### 計算規則
```
寒凍局（冬）：MB ∈ {亥, 子, 丑}
  → 最喜：丙丁巳午火，甲乙寅卯木，戊未戌土
  → 最忌：壬癸亥子水，庚辛申酉金
  → 調候急用：先取火暖局（調候為第一優先）

炎熱局（夏）：MB ∈ {巳, 午, 未}
  → 最喜：壬癸亥子水，庚辛申酉金，戊辰丑土（晦火）
  → 最忌：丙丁巳午火，甲乙寅卯木（助火）
  → 調候急用：先取水消暑（調候為第一優先）

濕木局（春）：MB ∈ {寅, 卯, 辰}
  → 溫度適中，不炎不凍，萬物有生機
  → 以日主強弱取用神為主，無需調候

燥金局（秋）：MB ∈ {申, 酉, 戌}
  → 溫度適中，不炎不凍，萬物有生機
  → 以日主強弱取用神為主，無需調候
```

### 輸出格式
```
【命局體性】X月生人，命局屬{寒凍/炎熱/濕木/燥金}。
{寒凍：最喜火木土暖局，最忌水金助寒。調候急用：丙丁火。}
{炎熱：最喜水金消暑，最忌火木助熱。調候急用：壬癸水。}
{濕木/燥金：體性溫和，以日主強弱論用神。}
```

## 第四章：格局與用神判定

### 輸入
- 月支(MB)、日主強弱結論、體性結論、全部八字

### 格局判定優先順序

#### 第一優先：夏冬季節 → 調候優先

```
若 MB ∈ {亥子丑巳午未}（寒凍/炎熱局）：
  炎熱局 → 優先以水為用神（調候優先），次選金
  寒凍局 → 優先以火為用神（調候優先），次選木土
  調候用神若有透出且成立（見下方透出條件）→ 直接定為用神
  若調候用神無法透出 → 退而採「第二優先」邏輯
```

#### 第二優先：月令旺相休囚死定格

```
月支藏干按旺相休囚死排序，取最旺者對應之十神定格：

月支藏干各干皆依「旺相休囚死乘數」計算力量：
  力量 = 藏干基礎點數（本氣5/中氣3/餘氣2）× 旺相休囚死乘數

若多干透出，比較各透出干的月支藏干力量，力量最大者優先定格。

常用格局：
  正官透 → 正官格
  七殺透 → 七殺格
  正印透 → 正印格
  偏印透 → 偏印格（梟神格）
  正財透 → 正財格
  偏財透 → 偏財格
  食神透 → 食神格
  傷官透 → 傷官格
  比劫月令旺 → 建祿格/月劫格

無透出者，取月支本氣藏干之十神定格。
```

#### 變格條件

```
從強格（棄命從強）：
  條件：身強度 ≤ 20（極弱），且命局無印比幫身，
        官殺/財星/食傷極旺（三項合計 ≥ 70%）
  喜：助旺五行方向
  忌：印比（違背從勢）

從旺格（一行/二行五行）：
  條件：身強度 ≥ 80（極強），且無官殺制，
        命局集中於一或二行五行（合計 ≥ 75%）
  喜：比劫/印星（順其旺勢）
  忌：官殺/財（破其旺勢）
```

### 第三優先：用神透出成立條件（三重驗證）

```
用神透出須同時滿足以下三條件，方視為「有效透出」：

條件一：天干透出
  用神五行必須出現在年干/月干/時干之一

條件二：左右無克（天干不被夾攻）
  透出用神天干，其左右相鄰天干中不得同時有克制之干
  例：用神甲木，若年干庚、月干甲、時干辛 → 甲木被庚辛左右夾克 → 條件不成立
  （僅單側有克，仍視為有效，需標注「稍有損傷」）

條件三：地支通根且根未被刑沖合害
  通根：用神五行對應地支（本氣或中氣）存在於四柱地支之一
  根的地支互動檢查：
    被六沖 → 根力減半（通根力×0.5），若為唯一根則視為通根不足
    被三刑 → 根力×0.6
    被六害 → 根力×0.7
    被六破 → 根力×0.75
    被六合化去他行 → 原根力消失
    有三合/三會加強 → 根力翻倍，通根最強

三條件皆成立 → 用神透出有力（標注：透干有力）
條件一成立、二三有瑕 → 用神透出有損（標注：透而不清）
條件一不成立 → 用神僅藏支（標注：藏而未透，力量受限）
```

### 用神選取邏輯

```
（優先順序）

1. 調候優先（體性極端時）：
   炎熱局 → 用神首選水，次選金
   寒凍局 → 用神首選火，次選木土

2. 扶抑法（正格）：
   身強（≥60%）：用神選官殺、財星（洩/克日主）
   身弱（≤40%）：用神選印星、比劫（生/助日主）
   中和（40-60%）：官財印均可，以格局成立者優先

3. 通關法：
   兩行五行對峙（如木金對峙，木%+金%≥70%）→ 用中間通關五行（水通關）

4. 病藥法：
   命局有「病」（破壞格局之字）→ 找「藥」（制病之字）為用神

喜用神確定後，推導：
  喜用神五行 → 喜天干（如水→壬癸）
  喜用神五行 → 喜地支（如水→亥子）
  忌神五行 → 忌天干
  忌神五行 → 忌地支
```

### 輸出格式
```
【格局】{正官格/七殺格/食神格/...} 或 {從強格/從旺格}
  定格依據：月令{旺/相/休/囚/死}（乘數×X），{透干名}透出
  若調候優先：（夏/冬月，調候優先）

【用神】{五行名}（理由：調候/扶抑/通關/病藥）
  透出狀態：{透干有力/透而不清/藏而未透}
  通根：{地支X}（{有無刑沖害破影響}）

【喜用】天干：{X X} 地支：{X X X}
【忌神】天干：{X X} 地支：{X X X}
【格局說明】...（該格局之一般特性與成就方向）
```

## 第五章：六親論斷（量化版）

> 每位六親以 0-100 分量化緣分，基準分 70，依刑沖克害加減。

### 六親緣分評分公式

```
基準分 = 70

加分項：
  對應柱為喜用神          → +15
  對應柱六合（地支）       → +10
  對應柱三合/三會（地支）  → +20
  對應柱生扶日主           → +10

扣分項（直接引用刑沖克害）：
  天干被克（對應柱干受克） → -15
  地支六沖                 → -20
  地支三刑                 → -15
  地支六害                 → -10
  地支六破                 → -8
  對應柱為忌神             → -15
  對應柱天克地沖命局       → -25

最終分數 = clamp(基準分 + Σ加分 + Σ扣分, 0, 100)

評級：
  80-100 → 緣深（相扶相助）
  60-79  → 緣中（有緣有份）
  40-59  → 緣薄（各自為命）
  20-39  → 緣弱（少助多阻）
  0-19   → 無緣（離散之象）
```

### 計算規則

#### 年柱（祖父母/父母/出身，0-15歲）
```
年干年支各自查十神 → 判喜忌 → 套入評分公式
年干年支相生 → 祖業興旺，父母和順
年干年支相克 → 干克支不利母，支克干不利父
```

#### 月柱（父母/兄弟姐妹/青年運，16-31歲）
```
月干月支各自查十神 → 判喜忌 → 套入評分公式
月支為月令，月干透出為用者 → 父母有力，兄弟得助
注：50-69歲必行沖克生月之運，此段多事多變
```

#### 日支（配偶）
```
日支查十神（男：財=妻；女：官殺=夫）→ 套入評分公式
日干日支相生 → 加分；相克相沖 → 依上方扣分項計算
```

#### 時柱（子女/晚年運，48-65歲）
```
時干時支各自查十神 → 判喜忌 → 套入評分公式
時柱生扶日主 → 子女賢孝；時柱沖日柱 → 子女緣薄
```

### 輸出格式
```
【年柱·出身祖業】{干支}為{十神}，緣分：XX分（{緣深/緣中/緣薄/緣弱/無緣}）
  刑沖害破：{無 / 有，說明影響}
  {出身論述：祖上條件、早年環境、父母狀況}

【月柱·父母兄弟】{干支}為{十神}，緣分：XX分（{評級}）

  {父母緣分、兄弟關係、青年運（16-31歲）}

【日支·配偶緣分】{支}為{十神}，緣分：XX分（{評級}）
  刑沖害破：{說明}
  {配偶條件、婚姻狀況、夫妻相處}

【時柱·子女晚運】{干支}為{十神}，緣分：XX分（{評級}）
```

```
日主五行性格基礎：

格局修正：

  七殺格 → 魄力、敢冒險、衝動
  食神格 → 享受、藝術、隨和
  傷官格 → 聰明、叛逆、才華橫溢
  正財格 → 務實、勤勞、重物質
```

## 第十章：大運逐運論斷（百分制評分）

> 每個大運10年，逐一計算評分，支援走勢圖輸出。

### 大運評分計算（每個大運）

#### Step 1：大運五行強度計算

```
大運基礎點數：
  大運天干：10 點
  大運地支本氣：5 點
  大運地支中氣：3 點
  大運地支餘氣：2 點
  大運總基礎 = 20 點

套用旺相休囚死乘數（依命局月支季節）：
  大運天干五行 × 旺相休囚死乘數
  大運地支各藏干五行 × 旺相休囚死乘數

大運五行分布（%）= 各五行調整後點數 / 大運總調整點數
```

#### Step 2：調候/用神喜忌評分基礎

```
優先判斷（依體性）：

[夏冬月（巳午未/亥子丑）→ 調候優先]
  炎熱局：大運中水%越高 → 越利（調候）
  寒凍局：大運中火%越高 → 越利（調候）
  調候分 = 調候喜用五行在大運中% × 100

[春秋月（寅卯辰/申酉戌）→ 直接用神喜忌]
  用神分 = 喜用神五行在大運中% × 100

大運基礎分 = 調候/用神分 - 忌神五行在大運中% × 100 + 50（基準線）
```

#### Step 3：地支互動加減分

```
大運支 與 命局四支 的互動：

加分：
  三合（大運支+命局兩支，合成喜用五行）  → +25，且三合五行點數 ×2
  三會（大運支+命局兩支，會成喜用五行）  → +30，且三會五行點數 ×2.5
  六合（大運支合命局某支，合去忌神）     → +15
  沖去忌神（大運支沖命局忌神地支）       → +10

扣分：
  三合（合成忌神五行）    → -25
  三會（會成忌神五行）    → -30
  六沖（沖去喜用神地支）  → -20
  六害（大運支害命局某支）→ -10
  六破（大運支破命局某支）→ -8
  三刑（大運支刑命局某支）→ -15
  天克地沖（大運干支均克命局日柱）→ -30
  歲運並臨（大運干支=流年干支，且為忌神）→ -20

特殊加分：
  歲運並臨（大運干支=流年干支，且為喜用）→ +20
```

#### Step 4：最終大運分數

```
大運分 = clamp(大運基礎分 + Σ地支互動加減分, 0, 100)
```

```
規則 B02：月柱天克地沖日柱
  → 兄弟感情不睦，各奔東西，難以相扶
  觸發條件：月干克日干 AND 月支六沖日支

規則 B03：月支被日支六沖
  → 與兄弟分離之象，少年時期即分居各地
  觸發條件：月支 六沖 日支

規則 B04：比劫% < 8%
  → 兄弟緣薄，孤立無援，少有兄弟助力
  觸發條件：比劫五行佔命局% < 8%
```

```
  → 比劫奪財，財路受阻，合夥易損，獨立經營較佳
  觸發條件：比劫% ≥ 30% AND 財星% < 15%

  → 靠才能/技術賺財，事業財路相輔相成，越做越順
  觸發條件：食傷%≥15% AND 財星%≥15% AND 日主能勝財
```
